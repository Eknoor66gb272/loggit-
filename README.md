# Logg IT | High Precision Wages

Premium industrial-grade work hour logging system with HQ Ledger and Personnel Registry.

**Production URL:** https://logg-it-high-precision-wages-786903666039.us-west1.run.app/

## Project Overview

Logg IT is a sophisticated payroll management suite designed for high-precision environments. It features:
- **Cloud-Linked Ledger**: Real-time synchronization with Supabase.
- **Security Check Protocol**: Mandatory passcode activation and identity verification.
- **Personnel Registry**: Advanced employee management with status archiving (login blocking).
- **Foundry UI**: An integrated design tool for building custom system components.

## Local Development

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
